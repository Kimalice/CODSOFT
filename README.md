Customer Churn Prediction
📊 Project Overview
This project focuses on predicting customer churn for a subscription-based service using machine learning techniques. The goal is to analyze customer data and identify those likely to discontinue their service, enabling businesses to implement strategies to improve customer retention.

🔍 Data Description
The dataset contains 10,000 customer records with the following features:

RowNumber: Row index
CustomerId: Unique identifier for each customer
Surname: Customer's surname
CreditScore: Customer's credit score
Geography: Customer's country of residence
Gender: Customer's gender
Age: Age of the customer
Tenure: Number of years the customer has been with the company
Balance: Customer's account balance
NumOfProducts: Number of products the customer has
HasCrCard: Whether the customer has a credit card (1 = Yes, 0 = No)
IsActiveMember: Whether the customer is an active member (1 = Yes, 0 = No)
EstimatedSalary: Estimated annual salary of the customer
Exited: Indicates if the customer has churned (1 = Yes, 0 = No)

📈 Model Development
Data Preprocessing: Handled missing values, transformed categorical variables, and scaled numerical features.
Exploratory Data Analysis (EDA): Visualized customer demographics and churn patterns to gain insights.
Model Training:
Logistic Regression: Achieved an accuracy of 81.1%.
Random Forest: Achieved an accuracy of 86.65%.
Evaluation Metrics: Used precision, recall, F1-score, and confusion matrix to assess model performance.
📊 Results
Logistic Regression:

Accuracy: 81.1%
Precision: 0.83 (Class 0), 0.55 (Class 1)
Recall: 0.96 (Class 0), 0.20 (Class 1)
Random Forest:

Accuracy: 86.65%
Precision: 0.88 (Class 0), 0.76 (Class 1)
Recall: 0.96 (Class 0), 0.47 (Class 1)
